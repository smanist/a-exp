# Task Schema

```markdown
- [ ] Imperative task title
  Why: Why this matters.
  Done when: Mechanical completion condition.
  Priority: high|medium|low
```

For mid-sized tasks, `Done when` may be a checklist:

```markdown
- [ ] Imperative task title
  Why: Why this matters.
  Done when:
  - Mechanical completion condition.
  - Another mechanical completion condition.
  Priority: high|medium|low
```

Completed tasks use `[x]`. Do not use partial completion annotations.
